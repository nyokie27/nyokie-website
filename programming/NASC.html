Nyokie's Artillery Support Calculator



0









How to use:
Green = Point A (starting position), Red = Point B (target position)
Set grid size (amount of tiles per row and collumn)
Set map scale (use meters)
Input loaded projectile's initial velocity (use meters per second squared)

Calculations are done automatically after two points are plotted. Clicking again will overwrite the previously selected target.
